How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'roller_authentication', github: 'dsandstrom/roller_authentication
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install roller_authentication
```
